## Getting Started

* Clone the repository

    ```sh
    git clone https://github.com/detailswes/cemboo-ott-reactjs
    curl -fsSL https://get.pnpm.io/install.sh | sh -
    pnpm install
    ```

* Project setup

    ```sh
    cd cemboo-ott-reactjs/
    pnpm install
    ```

* Start the development server

    ```sh
    pnpm run dev
    ```
